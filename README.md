# AI Posture Corrector 

**An AI-powered real-time posture analysis tool designed for desk workers.**

This project uses Computer Vision to detect poor posture (specifically "Turtle Neck" syndrome) and provides visual feedback to help users maintain a healthy sitting position.

## Features
- Real-time webcam feed analysis.
- Posture detection using **MediaPipe Pose Estimation**.
- Visual alerts for incorrect posture (e.g., Forward Head Posture).

## Tech Stack
- **Language:** Python 3.x
- **Libraries:** OpenCV, MediaPipe, NumPy
